# Приложение "Путевой Компас"

## Обзор

Это приложение на Next.js, разработанное как AI-помощник для планирования путешествий. Оно помогает пользователям создавать маршруты поездок и получать рекомендации по турам и жилью от искусственного интеллекта.

## Технологический стек

*   **Фреймворк**: Next.js (с App Router)
*   **Язык**: TypeScript
*   **UI**: React
*   **Стилизация**: Tailwind CSS
*   **UI Компоненты**: ShadCN UI
*   **AI**: Google Genkit
*   **Иконки**: Lucide React
*   **Карты**: Google Maps React (@vis.gl/react-google-maps)
*   **Формы**: React Hook Form и Zod

## Функционал

### 1. Управление маршрутами
*   **Создание маршрутов**: Пользователи могут создавать новые маршруты путешествий, указывая название, пункт назначения и даты поездки.
*   **Просмотр маршрутов**: На главной странице отображается список всех созданных маршрутов.
*   **Детали маршрута**: У каждого маршрута есть своя страница, на которой отображаются:
    *   Интерактивная карта с метками для всех мест в маршруте.
    *   Список мест (достопримечательности, рестораны и т.д.).
    *   Рекомендации от AI по добавлению новых мест в маршрут.

### 2. Рекомендации от AI
*   **Подбор туров**: На странице "Поиск Туров" пользователи могут найти туры и экскурсии на основе пункта назначения, дат и интересов. AI предоставляет список релевантныx активностей с описаниями и ссылками для бронирования.
*   **Подбор жилья**: Страница "Аренда Жилья" позволяет пользователям находить жилье. На основе пункта назначения, дат и предпочтений (бюджет, стиль) AI предлагает отели, апартаменты и т.д., с указанием плюсов, минусов и ориентировочной цены.
*   **Рекомендации мест**: В рамках конкретного маршрута AI предлагает уникальные места и занятия, соответствующие пункту назначения и интересам пользователя.

### 3. Пользовательский интерфейс
*   **Адаптивный дизайн**: Приложение корректно отображается как на настольных компьютерах, так и на мобильных устройствах.
*   **Современный UI**: Создан с использованием компонентов ShadCN UI для чистого и современного внешнего вида.
*   **Боковая панель**: Складная боковая панель для удобной навигации между разделами приложения.

## Как запустить приложение

Чтобы запустить сервер для разработки, выполните следующую команду в вашем терминале:
```bash
npm run dev
```
Приложение будет доступно по адресу `http://localhost:9002`.
